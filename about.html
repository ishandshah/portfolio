About Ishan Shah

Ishan Shah is a Staff Software Engineer based in San Jose, California, focused on distributed systems, cloud-native platforms, event-driven architecture, and real-time data engineering.

Engineering Background

Across PayPal, Nordstrom, and Securonix, Ishan has worked on systems where correctness, latency, reliability, and operational visibility matter. His experience includes building Java and Spring-based microservices, Kafka event pipelines, AWS cloud infrastructure, Kubernetes deployments, Terraform automation, real-time CDC systems, Solr search platforms, and observability practices for production engineering teams.

His work spans cloud-native billing and usage-based billing, real-time inventory platforms, Oracle RMS CDC to Kafka pipelines, fulfillment allocation optimization, supply chain technology, security analytics, Solr search infrastructure, and high-throughput backend services.

Ishan also contributes to technical leadership through architecture reviews, API contract discussions, schema governance, Kafka topic design, engineering mentorship, incident readiness, and AI-assisted software engineering practices for documentation, design validation, and code review.
